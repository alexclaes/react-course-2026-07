# React-Anwendungen debuggen

## Aufgabe 1: React Developer Tools installieren

Installiere die React Developer Tools als Browser-Erweiterung:

- **Chrome:** [React Developer Tools im Chrome Web Store](https://chromewebstore.google.com/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi)
- **Firefox:** [React Developer Tools bei Firefox Add-ons](https://addons.mozilla.org/firefox/addon/react-devtools/)
- **Microsoft Edge:** Über die [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons) nach „React Developer Tools" suchen.

## Aufgabe 2: Die React DevTools öffnen

1. Starte deine Message Board-App (`npm run dev`) und öffne sie im Browser.
2. Drücke **F12**, um die Browser-DevTools zu öffnen.
3. Öffne den **Components**-Tab (ggf. hinter dem **">>"**-Überlauf-Button).

## Aufgabe 3: Den Component-Baum erkunden

Sieh dir im Components-Tab den React-Component-Baum an:

```
App
  Header
  Post
```

## Aufgabe 4: Eine Component untersuchen

1. Klicke auf **Post** im Component-Baum.
2. Sieh dir im rechten Panel die **Props** und **Hooks** der Component an.
3. Klicke auch auf **Header** und **App**, um deren Details zu sehen.
