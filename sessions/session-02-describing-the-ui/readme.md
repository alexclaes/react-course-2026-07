# Session 02: Das UI beschreiben

## 💪 Motivation

Diese Session vertieft dein Verständnis der Komponentenhierarchie und des Datenflusses in React-Anwendungen.

Du lernst, Daten und Darstellung innerhalb von React-Komponenten zu verwalten, was für die Erstellung von UI unabdingbar ist.

## 🎯 Lernziele

- Tieferes Verständnis von React Component-Komposition
- Wissen, wie man Daten mit Props an React-Komponenten übergibt
- Verständnis des Datenflusses in React-Anwendungen
- Externe Pakete mit `npm` installieren und die UI-Bibliothek **Mantine** nutzen
- Die `children` Prop verwenden
- Komponenten als Props übergeben
- Daten aus Arrays mit `map` rendern
- Die `key` Prop anwenden
- Bedingtes Rendern mit dem `&&`-Operator
