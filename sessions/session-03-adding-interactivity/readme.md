# Session 03: Interaktivität hinzufügen

## 💪 Motivation

In dieser Session lernst du, State zu verwalten: Daten, die sich während der Lebensdauer einer Anwendung ändern.

Du wirst erkunden, wie Benutzerinteraktionen in React-Anwendungen zustandsbehaftete Daten aktualisieren können, was zu einem erneuten Rendern des UI führt.

## 🎯 Lernziele

- Auf Benutzerinteraktionen / Events in React reagieren, wie `onClick`
- Das Konzept von State verstehen
- Grundlagen über Hooks in React kennen
- Den `useState` Hook verwenden
- Grundlegendes Verständnis der State-Unveränderlichkeit
- Den Render-Zyklus in React kennen
