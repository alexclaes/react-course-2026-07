# Session 04: Komplexe Daten im State

## 💪 Motivation

Diese Session vermittelt dir die Fähigkeiten, komplexe Datenstrukturen wie Arrays und Objekte innerhalb von Reacts State-Management-System zu verwalten.

Du wirst Best Practices für das Rendern von Listen und das Verwalten von in Formularen eingegebenen Daten kennenlernen.

## 🎯 Lernziele

- Arrays im State speichern
- Formulare in React verwenden
- Formularübermittlung mit der FormData API verarbeiten
- Elemente zu einem im State gespeicherten Array hinzufügen
- Ein tieferes Verständnis der State-Unveränderlichkeit erlangen
- Controlled Inputs verstehen
