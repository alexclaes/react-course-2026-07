# Session 05: State verwalten

## 💪 Motivation

Diese Session konzentriert sich auf das Beherrschen von Techniken zum Teilen von Daten zwischen Components, was für die Erstellung komplexerer Anwendungen unerlasslich ist.

## 🎯 Lernziele

- Das Konzept des Datenaustausches zwischen Components verstehen
- Die Technik "Lifting State up" anwenden
- Nachteile von Prop Drilling kennen
- Elemente aus einem im State gespeicherten Array entfernen
- Context mit `createContext` erstellen und mit einem Provider bereitstellen
- Daten mit dem `useContext` Hook aus einem Context auslesen
