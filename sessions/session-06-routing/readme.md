# Session 06: Routing mit `react-router`

## 💪 Motivation

In dieser Session geht es um die Grundlagen zum Erstellen von Single Page Applications (SPAs) mit React.

Du lernst, wie du einzelne Seiten erstellst und zwischen ihnen navigierst, ohne die Seite neu zu laden, indem du clientseitiges Routing in der Message Board-App implementierst.

## 🎯 Lernziele

- Das Konzept von Single Page Applications (SPA) verstehen
- Clientseitiges Routing verstehen
- `BrowserRouter` einrichten
- Page Components erstellen
- Routes mit `Routes` und `Route` definieren
- Navigation mit `NavLink` und `Link`
- URL-Parameter mit `useParams` auslesen
- Programmatische Navigation mit `useNavigate`
