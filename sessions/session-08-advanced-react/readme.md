# Session 08: Weiterführende React Techniken

## 💪 Motivation

Du beherrschst die Basic von React und möchtest nun weiterführende Techniken lernen.

## 🎯 Lernziele

- Verwendung des `useReducer` Hooks
- Verwendung des `useCallback` Hooks
- Verwendung des `useMemo` Hooks
- Erstellen eigener Custom Hooks
- Verwendung des `useRef` Hooks
- Vermeiden unnötiger Re-Renders mit `React.memo`
- Tieferes Verständnis des `useEffect` Dependency-Arrays
